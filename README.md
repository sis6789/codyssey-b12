---
## 1. 브랜드 아이덴티티

- 브랜드명: 대전충만교회
- 타겟: 삶의 방향을 고민하는 20~30대 청년층
- 톤앤매너: 
	- 진지하고 따뜻함
	- 사실적인 도시 배경
	-  빛과 그림자의 대비
	-  잔잔하다가 벅차오르는 감정선
- USP(차별점): 고뇌에서 만남, 만남에서 회복으로 이어지는 실제적 변화를 강조
## 2. 캠페인 목표 / 핵심 메시지

- 캠페인 목적: 인지 + 전환 — 청년 대상 교회 첫 방문 유도
- 핵심 메시지(한 문장): "고민의 끝에서, 당신을 만나러 오신 분이 있습니다." 
## 3. 사용 도구 목록

- 이미지 생성:
	- nano Banana - 한글 처리가 탁월함
	- DALL·E - CoPilot에서 무료로 사용 가능
- 비디오 제작:
	- Gemini Omni - 프롬프트 반영력, 네이티브 오디오, 4K 출력에서 강점. 
- 오디오 생성: 
	- Gemini Omni — 영상을 제작하면서 적절한 음원을 제공함.
## 4. 씬 구성

- 기본 시나리오
	- 고뇌하는 청년이 예수님을 만나 대전충만교회로 출석
	- 5개 씬으로 구성
- 제작 방식
	- 각 씬을 대표하는 이미지를 제작한다.
	- 최종 영상은 5개 씬을 대표하는 이미지를 입력으로 하여 최종 10초 영상을 제작하도록 한다.
	- 각 씬의 연결은 fade-out-in을 적용한다.
	- 각 씬 내에서의 동적 효과는 영상 제작 AI 기능을 활용한다.

## 6. 제작 방식 및 일관성 유지 전략

- 씬 별 동영상 제작은 잔체 10초로 제한되는 조건에서 블가능함
	- 일반적으로 AI 동영상은 일정 시간 이상의 제작으로 되어 2초 3초 정도의 조정이 어려움
	- 이에 씬별 대표 이미지를 제작하고 이를 사용한 동영상 제작이 현 조건에 족합함
- 씬 별 이미지 생성이 독립적으로 수행되는 바, 청년의 이미지 유지가 필요함
	- 이를 위해 청년의 이미지를 타 씬 제작에 입력으로 하여 청년 이미지 특성을 유지함

## 7. 크레딧/도구 제약 대응 전략

- 동영상 제작은 상당한 크레딧을 요구함
- 이에 씬별 비디오 제작을 지양하고 이미지 생성으로 대체함
- 최종 동영상은 이 이미지에 동적 특성을 부여하는 AI의 기능을 사용함
- 10초 동영상에 최적화된 생성 전략을 세웠음.
